# UI Wireframes

## Overview

The AI Sports Injury Risk Detection System will provide a simple, user-friendly web interface for athletes, coaches, physiotherapists, sports scientists, and administrators.

The wireframes below describe the planned screens for the application.

---

# Screen 1 – Login Page

Components:

- Email
- Password
- Login Button
- Register Button
- Forgot Password

Purpose:

Allow users to securely access the platform.

---

# Screen 2 – Registration Page

Components:

- Full Name
- Email
- Password
- Confirm Password
- Select Role
- Register Button

Purpose:

Create a new user account.

---

# Screen 3 – Dashboard

Components:

- Sidebar Navigation
- Athlete Profile
- Upload Video
- Previous Reports
- Injury Risk Summary
- Notifications

Purpose:

Main dashboard after login.

---

# Screen 4 – Athlete Profile

Components:

- Name
- Age
- Gender
- Height
- Weight
- Sport Type
- Injury History
- Training Load

Purpose:

Store athlete information for accurate injury prediction.

---

# Screen 5 – Video Upload

Components:

- Upload Video Button
- Drag & Drop Area
- Upload Progress
- Video Preview
- Analyze Button

Purpose:

Upload sports activity videos.

---

# Screen 6 – Processing Screen

Components:

- Upload Status
- Progress Bar
- Frame Processing
- Pose Detection Status

Purpose:

Display AI analysis progress.

---

# Screen 7 – Injury Risk Report

Components:

- Risk Score
- Injury Type
- Confidence Score
- Joint Angles
- Detected Issues
- Download PDF

Purpose:

Display injury prediction results.

---

# Screen 8 – Recommendations

Components:

- Posture Correction
- Mobility Exercises
- Strength Exercises
- Recovery Plan
- Training Suggestions

Purpose:

Provide preventive recommendations.

---

# Screen 9 – Reports History

Components:

- Previous Reports
- Search
- Download PDF
- View Details

Purpose:

Access previous analyses.

---
